- Cloud computing: delivery of computing services over the internet.
	- Involves compute, networking, and storage.
	- Benefits:
		- On demand self-service.
		- Elasticity/Scaling
		- Reliability.
		- Less maintenance.

- Shared Responsibility Model (IaaS)
	- AWS takes care of (security of the cloud):
		- Software
			- Compute
			- Storage
			- Database
			- Networking
		- Hardware/AWS Global infrastructure
			- Regions
			- Availability zones
			- Edge locations
	- Customers of AWS have to take care of (security in the cloud):
		- Customer data.
		- Platform, applications, identity, and access management.
		- OS, network, firewall configs.
		- Client-side data encryption and data integrity authentication.
		- Server-side encryption.
		- Networking traffic protection.

- IaaS:
	- Infrastructure as a Service.
	- AWS, Cloudflare
- PaaS:
	- Platform as a Service
	- Firebase, Vercel, Heroku
- SaaS:
	- Software as a Service:
	- Gmail, ChatGPT, Discord

- Private vs Public Cloud
	- Private: Infrastructure provisioned for the use of one dedicated customer.
	- Public: Cloud infrastructure provisioned for public use.
